# SL铸币服务器官方网站系统

## 🎮 系统概述

这是一个功能完整的游戏服务器官方网站系统，专为SL铸币服务器设计。系统包含用户注册登录、公告管理、服务器规则、插件展示、管理人员介绍等核心功能，并采用现代化的设计风格和安全的架构设计。

## 🔧 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: PHP 7.4+
- **数据库**: MySQL 5.7+
- **设计**: 响应式设计，现代化UI

## 📋 主要功能

### ✅ 已实现功能
- **用户系统**: 注册、登录、登出、个人资料管理
- **公告系统**: 发布公告、编辑、删除、优先级管理
- **服务器规则**: 分类管理、违规等级、惩罚措施
- **插件展示**: 插件列表、版本信息、配置链接
- **服务器信息**: IP地址、端口、在线状态
- **管理人员**: 管理员列表、联系方式、职责介绍
- **安全功能**: CSRF保护、密码加密、登录限制
- **响应式设计**: 适配各种设备屏幕

### 🔒 安全特性
- **密码安全**: 使用bcrypt加密存储
- **会话管理**: 安全的会话处理机制
- **输入验证**: 严格的输入过滤和验证
- **CSRF保护**: 防止跨站请求伪造攻击
- **登录限制**: 防止暴力破解
- **错误处理**: 安全的错误信息显示

## 🚀 快速开始

### 1. 环境要求
- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- Apache/Nginx Web服务器
- 支持PDO扩展

### 2. 安装步骤

#### 方法一：使用安装向导（推荐）
1. 上传所有文件到您的Web服务器
2. 访问 `http://yourdomain.com/install.php`
3. 按照向导完成安装配置
4. 删除 `install.php` 文件
5. 使用默认管理员账户登录（用户名：admin，密码：admin123）

#### 方法二：手动安装
1. 创建MySQL数据库
2. 导入 `database.sql` 文件
3. 编辑 `config.php` 文件，配置数据库连接
4. 访问网站首页

### 3. 文件结构
```

├── index.html              # 主页面
├── config.php              # 配置文件
├── install.php             # 安装向导
├── database.sql            # 数据库结构
├── css/
│   └── style.css           # 样式文件
├── js/
│   └── main.js             # 主JavaScript文件
├── api/
│   ├── auth.php            # 用户认证API
│   └── content.php         # 内容管理API
└── README.md               # 说明文档
```

## 🎨 自定义设计

### 颜色主题
系统采用金色主题，您可以在 `css/style.css` 中修改CSS变量来自定义颜色：
```css
:root {
    --primary: 45 100% 50%; /* 金色 */
    --secondary: 210 100% 50%; /* 蓝色 */
    /* 更多颜色变量... */
}
```

### 布局调整
所有样式都集中在 `css/style.css` 文件中，您可以：
- 修改颜色方案
- 调整字体大小
- 更改布局结构
- 添加动画效果
